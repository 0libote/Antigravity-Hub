Documentation

Welcome to the official documentation for Antigravity Hub, the premium local-only UI extension for tracking your AI usage natively in VS Code.

Note: Antigravity Hub is a frontend UI. It requires the base Antigravity extension to be active to fetch real quota data.

Installation

You can install Antigravity Hub directly from the VS Code Marketplace or build it from source.

From Source

git clone https://github.com/0libote/AntiGravity-Hub.git
cd AntiGravity-Hub
npm install
npm run compile
# Press F5 in VS Code to run in Extension Development Host

Extension Settings

Antigravity Hub is highly configurable via VS Code Settings or through the in-dashboard UI modal.

  • antigravityHub.pollInterval: Time in ms between local API checks (Default: 500ms).
  • antigravityHub.statusBarFormat: Choose between Standard, Compact, Minimal, or Percentage.
  • antigravityHub.warningThreshold: When to turn visual indicators yellow (Default: 60%).
  • antigravityHub.criticalThreshold: When to turn visual indicators red and send alerts (Default: 20%).

Dashboard Styles

The main webview supports multiple layouts depending on how much detail you want.

  • Group: Circular Gauges - Beautiful grouped donut charts representing time remaining.
  • Group: Line Graphs - Clean, horizontal progress bars.
  • Models (Grouped) - See individual models nested under their parent groups.
  • Models (Flat) - A dense grid of every model currently tracked.

FAQ

Why isn't my quota updating?

Make sure the base Antigravity proxy is running. If the Hub shows a red disconnected status in the top right, it means the local fetcher cannot connect to the backend.
